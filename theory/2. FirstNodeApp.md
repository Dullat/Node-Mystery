### **Whatâ€™s Happening Here?**

- `http.createServer()` â†’ Creates a server that listens for requests
    
- `(req, res)` â†’ Objects representing **incoming request** & **outgoing response**
    
- `res.statusCode` â†’ Sets the HTTP status
    
- `res.setHeader()` â†’ Sets response headers (like content type)
